JavaScript

De acuerdo a MDN: JavaScript es un lenguaje interpretado basado en scripts, es empleado para agregar elementos dinámicos a las páginas web. Es una tecnología que se ejecuta en el lado del cliente, es decir, en el navegador, y este se encarga de interpretar el código. Por ello, es más simple y fácil de entender que un lenguaje de alto nivel. Es oportuno mencionar, que cuenta con mucha documentación en la web, y se acopla perfectamente con los diversos gestores de contenidos del mercado.

El programa fue creado por Brendan Eich en la empresa Netscape Communications, y apareció bajo el nombre LiveScript. Luego, fue relanzado con el nombre JavaScript en una alianza con Sun Microsystem.
