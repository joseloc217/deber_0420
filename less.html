LESS

De acuerdo a su página web oficial: Less (que significa Hojas de Estilo Delgadas) es una extensión de lenguaje compatible con versiones anteriores para CSS. Esta es la documentación oficial de Less, el lenguaje y Less.js, la herramienta de JavaScript que convierte sus estilos Less en estilos CSS. Debido a que Less se parece a CSS, aprenderlo es muy fácil. Less solo hace algunas adiciones convenientes al lenguaje CSS, que es una de las razones por las que se puede aprender tan rápido.

Less es un preprocesador del lenguaje de hoja de estilo. Eso quiere decir que cualquier código CSS es automáticamente también un código Less válido (pero no al revés). El objetivo de Less es hacer más eficiente la escritura en código CSS. Para ello, este lenguaje influido por SASS ofrece diferentes complementos a los comandos CSS tradicionales como, por ejemplo, variables y funciones, las cuales, entre otras cosas, facilitan el trabajo con las hojas de estilo y permiten prescindir de la engorrosa duplicación de códigos.
